# User List App

React Native app with:

- Paginated users list (5 per page)
- Search by name
- Infinite scroll + pull-to-refresh
- User detail screen (name, username, email, phone, address, website)
- Redux Toolkit state management

## Setup

```bash
npm install
cd ios && pod install && cd ..
```

## Run

```bash
npm start
npm run android
npm run ios
```
